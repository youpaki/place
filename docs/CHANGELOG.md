# Changelog

## [1.0.0] - 2026-06-17

### Added
- Initial release
- Stereo VST3 effect plugin for vocal space creation
- Mid/Side processing architecture
- SIZE MAXIMIZER parameter (0-100%)
- SIDE BASS REMOVER parameter (0-100%) with 24 dB/oct Linkwitz-Riley high-pass
- Static mode with fixed 6-band mid channel EQ
- Vocal Follow mode with 6-band dynamic EQ driven by sidechain
- Side compensation EQ for stereo width preservation
- Stereo sidechain input with automatic host detection
- Full parameter automation support
- State save/load with perfect project recall
- Dark purple minimal UI with vector graphics
- HiDPI support and resizable interface
- Zero latency processing
- Realtime-safe, no heap allocations in audio thread
- Windows VST3 installer via Inno Setup
